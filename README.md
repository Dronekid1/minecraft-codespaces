# Minecraft Java Edition in GitHub Codespaces

Run Minecraft Java 1.21 in your browser!

## Quick Start

1. Click Code → Codespaces → Create codespace
2. Wait for build (5-10 minutes first time)
3. Go to PORTS tab → Click globe icon on port 6080
4. In terminal run: `./setup-minecraft.sh`
5. In desktop terminal run: `cd /workspace/minecraft && ./minecraft-launcher`
6. Login and play!

## Tips
- Set graphics to Fast
- Render distance: 8-12 chunks
- Close Codespace when done to save hours
